# RL-Algorithms
Reinforcement learning algorithms

# Introduction & Motivation
Project bao gồm triển khai tối thiểu (minimal implementation) của 1 số thuật toán RL. Khi tôi đọc các thuật toán RL mới (đặc biệt là sau 1 thời gian dài không code về RL), tôi không thể nhớ các thuật toán cơ bản và sẽ mất nhiều thời gian để nhớ lại các thuật toán này. Các nguồn tham khảo (Github, blog, paper) cũng dùng các biến thể khác nhau và triển khai với phong cách khác nhau làm tôi khó nhớ lại hơn. Tôi triển khai các thuật toán này để dễ nhớ và tận dụng lại khi tìm hiểu và thử nghiệm các project RL mới.
